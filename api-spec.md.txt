# Спецификация API: Пользователи

## GET /api/v1/users
Возвращает список всех пользователей.

Ответ: 200 OK — массив пользователей в JSON.

## POST /api/v1/users
Создаёт нового пользователя.

Тело запроса:
{
  "name": "Имя",
  "email": "email@example.com"
}

Ответ:
- 201 Created — пользователь создан
- 400 Bad Request — ошибка в данных
