# Функциональные требования

1. Система должна позволять создавать учётную запись пользователя.
2. Система должна позволять входить в учётную запись по email и паролю.
3. Система должна отображать список товаров с фильтрацией по категориям.

# Нефункциональные требования

1. Система должна отвечать на запросы не дольше 200 мс.
2. Система должна поддерживать 1000 одновременных пользователей.
